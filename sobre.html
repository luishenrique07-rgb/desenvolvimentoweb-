Objetivo

Este material foi criado para praticar apenas HTML, sem CSS e sem JavaScript.

A ideia é entender primeiro a estrutura e as tags mais comuns.

Tags importantes que você já viu

  1. <h1> até <h6> para títulos
  2. <p> para parágrafos
  3. <a> para links
  4. <ul>, <ol> e <li> para listas
