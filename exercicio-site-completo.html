Enunciado: Site Completo em HTML

Projeto para iniciantes: criar um mini site com 3 páginas.

Início | Exercícios

Tema do projeto

Meu Portfólio de Estudante

Objetivo

Criar um mini site com três páginas conectadas entre si: index.html, sobre.html e contato.html.

Arquivos obrigatórios

  1. index.html (Home)
  2. sobre.html (Sobre)
  3. contato.html (Contato)

Regras obrigatórias

  1. Usar apenas HTML (sem CSS e sem JavaScript).
  2. Todas as páginas devem ter header, nav, main e footer.
  3. O menu deve ter links para as três páginas e funcionar corretamente.
  4. Usar pelo menos dois títulos (h1 e h2).
  5. Usar pelo menos três parágrafos.
  6. Usar pelo menos uma lista (ul ou ol).
  7. Usar pelo menos uma imagem com atributo alt.
  8. Usar pelo menos um link externo com target="_blank".
  9. Na página de contato, criar formulário com nome, e-mail, mensagem e botão enviar.

Conteúdo sugerido por página

1) Home (index.html)

  • Uma apresentação curta sobre você.
  • Uma lista com o que existe no seu site.
  • Uma imagem representando seus estudos.

2) Sobre (sobre.html)

  • Quem é você.
  • Seus objetivos com tecnologia.
  • Um link externo para um site

3) Contato (contato.html)

  • Formulário com:
  • Campo Nome.
  • Campo E-mail.
  • Campo Mensagem.
  • Botão Enviar.
